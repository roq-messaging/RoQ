EQS

An EQS is a distributed message queue designed for clouds and that enables a linear scalability when new nodes are added to the cluster. As a result, the number of publishers-subscribers and the message load are not a limit anymore !

EQS was first presented at the IEEE 2011 CloudCom conference.
